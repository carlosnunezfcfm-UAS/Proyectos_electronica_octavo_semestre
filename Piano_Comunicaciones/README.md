El objetivo principal de este proyecto es diseñar e implementar un Oscilador Controlado por Voltaje (VCO), cuya frecuencia pueda modificarse mediante el uso de botones. Cada botón aplica un nivel de voltaje diferente al circuito, provocando un cambio en la frecuencia de oscilación y permitiendo generar distintas notas musicales.
Como reto adicional, se desarrolló un sistema de adquisición y visualización de señales con funciones similares a las de un osciloscopio. Este sistema permite observar en tiempo real las señales generadas por el integrador y el disparador Schmitt Trigger que conforman el oscilador.

En la imagen se muestran ambas señales después de la etapa de acondicionamiento. Como puede observarse, los niveles de voltaje fueron adaptados al rango de 0V a 5V, permitiendo su adquisición segura por parte del microcontrolador. Gracias a este proceso, las señales pueden ser digitalizadas, enviadas a la computadora y visualizadas mediante la interfaz desarrollada en Python, funcionando como una alternativa al uso de un osciloscopio convencional para fines de análisis y monitoreo del sistema.
